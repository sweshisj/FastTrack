# Claims FastTrack

## Problem Statement
The claims department currently relies on manual data entry to transfer claim information from customer intake forms into a legacy portal. This process is time-consuming, prone to human error, and lacks a centralized audit trail.

## Solution
Claims FastTrack is a web application that automates the intake and triage process. It provides:
1.  **Centralized Intake:** A modern web interface for submitting claims.
2.  **Automated Triage:** Validation rules to immediately identify incomplete or high-risk claims.
3.  **Human-in-the-loop Exception Management:** A queue for staff to review and fix flagged claims.
4.  **RPA Integration:** A bot automatically updates the legacy portal for valid claims, leaving a full audit log.
